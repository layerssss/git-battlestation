git-battlestation
=================

##功能：

### 包含gitweb中的基本功能

### 全局管理remote,快速从指定的remote,分支pull

* 简化merge信息

### 启动git-daemon

### 查看本地和某一个remote上的分支进展

列出所有其他分支上有的而主分支没有的提交。（链接到gitweb上对应的commitdiff）

### 快速查找、生成命令

* 撤销本地的某个提交： git reset HEAD~N --soft
* 单独截取某一个提交： git cherry-pick BRANCH COMMIT ...
